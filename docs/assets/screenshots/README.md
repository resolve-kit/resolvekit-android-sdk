# Screenshot Placeholders

Replace these placeholder files with actual screenshots from the sample app.

## Required files

| File | Description | Suggested resolution |
|---|---|---|
| `chat-view.png` | Main chat screen showing agent conversation | 1080×2280 |
| `tool-approval.png` | Approval prompt for a tool call requiring user consent | 1080×2280 |
| `activity-integration.png` | Full-screen ResolveKitChatActivity launched from host app | 1080×2280 |

## How to capture

1. Build and install the sample app on a device/emulator
2. Navigate to each demo screen
3. Use Android Studio's screenshot tool or `adb exec-out screencap -p > file.png`
4. Place images in this directory
5. The README.md table at the top of the repo root will display them
